Dark version for the default Cookie Notice theme

Default Dark - Theme Setup
  • Include default <div> and <script>
  • Download theme CSS file
  • Upload theme .css file to your website
  • Include the CSS in your document <head> section

Local self-hosted setup example:

<link rel="stylesheet" href="dist/css/themes/default-dark/cookienotice.min.css" />

OR

Include the hosted version in your <head> section:

<link rel="stylesheet" href="https://cookienotice.js.org/themes/default-dark/cookienotice.min.css" />

Soft Dark - Theme Setup
  • Include default <script>
  • Download theme CSS file
  • Upload theme .css file to your website
  • Include the CSS in your document <head> section

Local self-hosted setup example:

<link rel="stylesheet" href="dist/css/themes/soft-dark/cookienotice.min.css" />

OR

Include the hosted version in your <head> section:

<link rel="stylesheet" href="https://cookienotice.js.org/themes/soft-dark/cookienotice.min.css" />

Add the Cookie Notice <div> block into your <body>:

<div id="cookie-notice">
	We use cookies to deliver better experience.
	<a href="https://cookienotice.js.org/about-cookies" class="cookie-notice-more" target="_blank" rel="noopener">More info</a>
	<a href="javascript:void(0)" class="cookie-notice-close" onclick="closeCookieNotice()">×</a>
</div>
